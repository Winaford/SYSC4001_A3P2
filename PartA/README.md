Project:
Testing concurrency by simulating teaching assistants grading a stack of exams (WITHOUT SEMAPHORES)

Targets:
x86_64 Linux and POSIX compliant systems

Dependencies:
GCC / g++ compiler
make 

Build Instructions:
navigate to project dir and run "make" in terminal 

Running Instructions 
run "chmod +x grading_sim" 
then run ./grading_sim <ta_number> <exams_dir> <rubric_file>
